MikroTik Hotspot Voucher App - Simple Install

1. Make sure Python 3 is installed.
   Download: https://www.python.org/downloads/
   Tick "Add python.exe to PATH" during installation.

2. Open the main app folder and double-click INSTALL.bat.

3. After installation, open the app using:
   - Desktop shortcut: MikroTik Hotspot Vouchers
   - Start Menu: MikroTik Hotspot Vouchers
   - START_APP.bat

   The local server also starts automatically after Windows sign-in.

4. The app opens in your browser at:
   http://127.0.0.1:8008
   If that port is busy, START_APP.bat opens the next free port automatically.

5. In the app, open Router Settings and enter your MikroTik API details.

Auto-start:
- To enable manually, double-click ENABLE_AUTOSTART.bat.
- To disable, double-click DISABLE_AUTOSTART.bat.
