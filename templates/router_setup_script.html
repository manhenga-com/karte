Adoption Steps

  1. Create the WireGuard server on the Karte VPS and enter its public key, public endpoint, and tunnel IP above.
  2. Assign this router a unique tunnel address, such as 10.10.10.2/32.
  3. Change the default API password, then select Generate Adoption Script.
  4. Open the router in WinBox, open New Terminal, and paste the script.
  5. Copy the router public key printed in the terminal and add it as a peer on the VPS.
  6. Add the router in Karte with its tunnel IP without /32, API port, API username, and API password.
