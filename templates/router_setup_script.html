Router Setup Script

For a VPS-hosted app, use WireGuard mode. The router connects to the VPS, then the app manages the router using its WireGuard IP.

Copy This Script

How To Use

  1. On the VPS, create a WireGuard server and copy its public key into this page.
  2. Give each router its own Router WireGuard address, for example 10.10.10.2/32, 10.10.10.3/32, and so on.
  3. Open WinBox.
  4. Open New Terminal.
  5. Paste the script.
  6. Copy the router public key printed by the script and add it as a peer on the VPS with the router WireGuard address.
  7. In this app, add the router using its WireGuard IP without /32, for example 10.10.10.2.
